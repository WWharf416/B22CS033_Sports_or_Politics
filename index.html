Project Overview

Problem Statement

This project develops an automated system to classify news articles into two categories: Sport and Politics. Using machine learning techniques, the system can accurately predict the category of any news article based on its textual content.

Why This Matters

  • News Aggregation: Automatically organize articles by topic
  • Content Recommendation: Suggest relevant articles to users
  • Media Monitoring: Track coverage across different categories
  • Digital Archives: Automatically catalog documents

Key Achievements

  • ‚úÖ Perfect Classification: 11 out of 20 model-feature combinations achieved 100% accuracy
  • ‚úÖ Multiple Approaches: Compared Bag of Words, TF-IDF, and N-gram features
  • ‚úÖ Diverse Models: Tested Naive Bayes, Logistic Regression, SVM, Random Forest, and KNN
  • ‚úÖ Production Ready: Fast inference time (under 3ms) with perfect accuracy

Data Collection

Collected 928 news articles from the BBC News dataset, split into:

  • Training: 742 articles (80%)
  • Testing: 186 articles (20%)
  • Sport: 511 total articles
  • Politics: 417 total articles
üîß

Text Preprocessing

  • Lowercasing all text
  • Removing special characters
  • Tokenization using NLTK
  • Stopword removal
  • Lemmatization (e.g., "running" ‚Üí "run")

Feature Extraction

Three different approaches tested:

  • Bag of Words: Basic word frequency (1000 features)
  • TF-IDF: Weighted by importance (1000 features)
  • N-grams: Bigrams (1500) and Trigrams (2000)

Machine Learning Models

  • Naive Bayes: Probabilistic classifier
  • Logistic Regression: Linear model
  • SVM: Maximum margin classifier
  • Random Forest: Ensemble method
  • KNN: Instance-based learning

Key Insights

üéØ TF-IDF Superiority

TF-IDF features enabled three different model types to achieve perfect classification, while Bag of Words achieved this with only Naive Bayes.

‚ö° Speed vs Accuracy

Naive Bayes achieved 100% accuracy in just 0.002 seconds, making it ideal for real-time applications requiring instant classification.

üìà N-gram Impact

Bigrams improved KNN from 99.46% to 100%, but trigrams provided no additional benefit, showing diminishing returns.

üî¨ Model Consistency

Average accuracy of 98.82% across all 20 combinations with only 2.98% standard deviation demonstrates robust performance.
